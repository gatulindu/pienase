# Pienase 🍍
Pienase is Simples Minimal Server For You Aplication 💻

## How To Use 😺

Pass a HTML file it's run in `localhost:8080`
```
pie best.html
```

# Install

Runs a `build.sh`

```
# Auto Install
bash build.bash --auto
# Manual Install
bash build.bash
Compiler? clang
Install Dir? /bin/bestapps
```
